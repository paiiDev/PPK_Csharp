# 💻 C# Programming Introduction



## 📌 What is C#?



C# (C-Sharp) is a modern, object-oriented programming language developed by Microsoft. It is mainly used to build:



* Windows applications
* Web applications
* Games (using Unity)
* APIs and backend services



## 🎯 Why Learn C#?



* Easy to learn (similar to Java/C++)
* Strongly typed and type-safe
* Supports Object-Oriented Programming (OOP)
* Large ecosystem with .NET



##### ⚙️ Basic Structure of a C# Program



using System;



class Program

{

&nbsp;   static void Main(string\[] args)

&nbsp;   {

&nbsp;       Console.WriteLine("Hello, World!");

&nbsp;   }

}



###### 🔍 Explanation



* using System; → Imports built-in functionalities
* class Program → Defines a class
* Main() → Entry point of the program
* Console.WriteLine() → Prints output



## 🧱 Key Features of C#



##### **1. Object-Oriented**



Supports:



* Encapsulation
* Inheritance
* Polymorphism
* Abstraction



##### **2. Strongly Typed**



Every variable must have a defined type:



int age = 20;

string name = "John";



##### **3. Type Inference (var)**



var number = 10;  // compiler infers int



##### **4. Garbage Collection**



Automatic memory management (no need to manually free memory)





## 🔢 Data Types in C#
